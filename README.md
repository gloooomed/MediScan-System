# MediScan System

MediScan System is an intelligent, automated diagnostic interface designed to aggregate clinical parameters and provide preliminary probabilistic medical analysis.

## Core Capabilities
- Intuitive Input: Streamlined capture of symptoms through a structured clinical interface.
- Instant Compute: Harnesses proprietary neural pipelines for immediate predictive processing.
- High Fidelity: Maintains strict clinical accuracy parameters mimicking traditional diagnostics.
- Secure Execution: Enterprise-grade local execution for handling sensitive patient permutations.

## Development Setup

To begin local development on the MediScan System:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server (runs both frontend and backend concurrently):
   ```bash
   npm run dev
   ```
